Welkom op deze pagina! Hier ga ik je leren hoe je jou eigen bestand online moet krijgen! Hiervoor moet je wel eerst je site gocodeert hebben. Dit nog niet gedaan? Ga dan hierheen.

1) Druk op deze link om Github Pages te downloaden. Maak een account aan als je deze nog niet hebt.

2) Maak een mapje ergens waar je wil. Onthoud wel waar je deze neer zet en hoe je hem noemt!

3) Open GitHub Desktop en druk op Create new repository. Voer dan de naam in en kies de locatie waar je de repository wil neer zetten!

3) Als het goed is ziet hij dan dat er bestanden verandert zijn. Voer iets in bij het vakje onderin en druk dan op "commit to master".

5) Als je dat gedaan hebt druk je op "publish repository". Hiermee zorg je ervoor dat hij online gaat. Druk op "make repository public" en dan publish.

5) Ga dan naar je github pagina: https://www.github.com/(gebruikersnaam) <- Voer hier je naam in die hebt gebruikt om github aan te maken.

6) Ga dan naar "my profile" en druk op de repository met de naam die hebt gebruikt. Ga dan naar "settings" en scroll dan naar het kopje "Github pages".

7) Druk dan bij "source" op "master branche" en dan op save.

8) Ga dan naar het kopje "code" en druk dan op "edit". Voer dan bij site in: https://(gebruikersnaam).github.io

9) Feli je site staat online!
